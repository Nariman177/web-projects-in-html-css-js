html секциясы

HTML (ағылш. HyperText Markup Language — "гипермәтіндік белгілеу тілі")-браузерде веб-беттерді қарау үшін құжаттардың стандартталған гипермәтіндік белгілеу тілі. HTML гипермәтіндік белгілеу тілін британдық ғалым Тим Бернерс-Ли шамамен 1986-1991 жылдары Швейцариядағы Женевадағы CERN қабырғасында жасаған. HTML ғылыми және техникалық құжаттамамен алмасу тілі ретінде құрылды, оны орналасу мамандары емес адамдар қолдана алады. HTML тілінің алғашқы тегі Интернетте алғаш рет 1991 жылдың соңында Тим Бернерс-Ли айтқан «HTML тегтері» құжаты болды. Ол бастапқы, салыстырмалы түрде қарапайым HTML дизайнын құрайтын 18 элементті сипаттайды.
